# DoublyLinkedList

## Time Complexity
| Operation        | Time Complexity   |
|------------------|-------------------|
| **Append**       | O(1)              |
| **Prepend**      | O(1)              |
| **Insert**       | O(n)              |
| **Remove Last**  | O(1)              |
| **Remove First** | O(1)              |
| **Remove**       | O(n)              |
| **Get**          | O(n)              |
| **Set**          | O(n)              |
| **Print List**   | O(n)              |
