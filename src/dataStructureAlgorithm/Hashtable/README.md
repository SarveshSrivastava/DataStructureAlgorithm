# HashTable

## Hash Table Time Complexity

| Operation                           | Average Case | Worst Case |
|-------------------------------------|--------------|------------|
| **Insertion (`set`)**               | O(1)         | O(n)       |
| **Search (`get`)**                  | O(1)         | O(n)       |
| **Deletion (`remove`)**             | O(1)         | O(n)       |
| **Traversing all keys (`allKeys`)** | O(n)         | O(n)       |
