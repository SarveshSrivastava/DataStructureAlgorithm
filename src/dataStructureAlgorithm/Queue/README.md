# Queue

## Queue Time Complexity

| Operation       | Time Complexity|
|-----------------|----------------|
| **Enqueue**     | O(1)           |
| **Dequeue**     | O(1)           |
| **Get First**   | O(1)           |
| **Get Last**    | O(1)           |
| **Get Length**  | O(1)           |
| **Print Items** | O(n)           |
